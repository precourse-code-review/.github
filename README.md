### ⏰ 타임어택 (매주 월요일) - 필수

매주 월요일 1시 ~ 6시 지난 기수 문제를 풀어봅니다.

월요일에 제한시간을 두어서 푼 후,
수요일자정까지 리팩토링하여 아래 PR링크를 업데이트합니다. 

이후 일요일자정까지 피어리뷰를 갖도록 합니다.
